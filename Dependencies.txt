Python:
- You might want to use `pip3 install ...` instead of `pip install ...` depending on how your installation is set up.
- Beautiful Soup 4 (pip3 install beautifulsoup4)
- PIL: Python Imaging Library (pip install pillow)
- Pymongo: For Database (pip install pymongo)
- Dateutil : Date Parsing (pip install python-dateutil)
- Pytz: For timezones (pip install pytz)
